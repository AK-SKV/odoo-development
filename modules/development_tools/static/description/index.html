Development Tools

A small set of useful tools to analyze the Odoo behavior and make easy the Odoo module development.

Git project

The module Development Tools is part of a larger project that collect several items related with the development in Odoo.

It is hosted entirely on Github, it can be downloaded and used it for free under Affero GPL 3 license.

Domain and context tester

You will can choose any existing model view and show it applying any domain you have written.

The domains test assistant also allows updating the values of the context that will required by the view.

Python code tester

You can write your own python code expressions in a form view and execute it inside the Odoo context, getting the results in the same view.

Another option that is also available in the wizard is run the python code inside a server action.

Run actions from its own views

You will can execute Odoo actions from its own predefined views using new added option in button more. Several actions of the same type will can be executed together from the tree view.

Capture the outgoing emails

You will can capture all outgoing emails and redirect them to your own configurable email address.

Pivot tables with groups information

Relationships between groups, users and access rights

Groups can form hierarchies where items on top involve all the items which are below. Thereby, an user which belongs to a group also belongs to implied groups and this is not easy to see it.

Group inheritance

There is one report to show the relationships between groups.

Users by group

There is one report to show the relationships between groups and users.

Models by group

There is one report to show the model access rights by each group.
